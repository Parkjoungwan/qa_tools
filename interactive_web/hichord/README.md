# HiChord - 인터랙티브 웹 뮤직 앱

HiChord는 웹 브라우저에서 동작하는 인터랙티브 음악 제작 도구입니다. 사용자는 키보드와 마우스, 터치를 이용해 다양한 화음을 연주하고, 이를 녹음하여 자신만의 루프를 만들 수 있습니다.

## 주요 기능

- **7-패드 인터페이스**: `q, w, e, r, 2, 3, 4` 키에 매핑된 7개의 패드로 화음 및 드럼 샘플을 실시간으로 연주합니다.
- **실시간 화음 생성**: 선택된 조성(Key), 모드(Mode), 음질(Quality), 자리바꿈(Inversion)에 따라 음악 이론에 기반한 화음이 자동으로 생성됩니다.
- **8방향 프리셋**: `u, i, o, j, l, m, ,, .` 키를 이용해 8개의 다른 화음/드럼 설정으로 빠르게 전환할 수 있습니다.
- **로우파이 드럼 머신**: 킥, 스네어, 하이햇 등 다양한 사운드를 포함한 내장 드럼 머신 기능이 있습니다.
- **멀티트랙 루퍼**: 연주를 녹음하고 오버더빙(겹쳐서 녹음)하여 여러 트랙을 쌓아 올릴 수 있습니다.
- **고급 오디오 기능**:
  - **사이드체인 컴프레션**: 킥 드럼이 나올 때 다른 악기 소리가 살짝 작아져 펀치감을 더합니다.
  - **하이햇 초크**: 열린 하이햇과 닫힌 하이햇 사운드가 자연스럽게 끊어지도록 제어합니다.
  - **보이스 리딩**: 화음이 변경될 때 이전 음과 가장 가까운 음으로 부드럽게 연결됩니다.
- **다양한 입력 지원**: 키보드뿐만 아니라 마우스 클릭 및 터치 입력으로도 연주할 수 있습니다.
- **반응형 UI**: 데스크톱과 모바일 등 다양한 화면 크기에 대응합니다.

## 실행 방법

이 애플리케이션은 브라우저 보안 정책으로 인해 로컬 웹 서버를 통해 실행해야 합니다.

1.  프로젝트 폴더에 포함된 `serve.py`를 이용해 서버를 실행합니다. 터미널에 아래 명령어를 입력하세요.
    ```shell
    python3 serve.py
    ```

2.  서버가 실행되면 웹 브라우저를 열고 아래 주소로 접속합니다.
    *   **http://localhost:8000/hichord.html**

## 컨트롤

- **패드 연주**: `q`, `w`, `e`, `r`, `2`, `3`, `4`
- **프리셋 변경**: `u`, `i`, `o`, `j`, `l`, `m`, `,`, `.` (`,` = 드럼 모드)
- **녹음**: `Spacebar` (시작 / 정지)
- **트랙 제어**: `z`, `x`, `c`, `v` (트랙 활성화/비활성화). 해당 키를 2초간 길게 누르면 트랙이 삭제됩니다.
- **화음 변경**:
  - **조성(Key)**: `←` / `→` (한 음씩), `Shift` + `←` / `→` (한 옥타브씩)
  - **자리바꿈(Inversion)**: `↑`
  - **음질(Quality)**: `↓` (3화음/7화음 전환)
- **OP-KO 드럼 컨트롤**:
  - **뱅크**: `1`–`4` (또는 패널의 BANK 1–4 버튼)
  - **펀치-인 FX**: `A, S, D, F, G, H, J, K` (누르는 동안 효과 발동)
  - **FX 노브**: 패널의 각 FX 슬라이더로 실시간 강도 제어
  - **샘플 트림**: Slot 선택 후 Start/End/Reverse 조절
  - **스텝 리피트**: 16스텝 그리드에 비트를 찍어 루프에 맞춰 자동 재생
